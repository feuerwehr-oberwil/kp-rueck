# Task 3.3: Combined Kanban & Map View

## Overview

Create a new combined view that displays the Kanban board and map side-by-side for large screens, allowing users to see both tactical board and geographical overview simultaneously. This enhances situational awareness by providing both workflow status and spatial context in a single view.

**Estimated Time**: 8-12 hours

## Prerequisites

- ✅ Task 3.1: Geocoding Service (incidents have coordinates)
- ✅ Task 3.2: Map View UI (map component exists)
- ✅ Task 2.3: Kanban Board UI (kanban board exists)
- ✅ Synchronization features (highlight param, map markers clickable)

## Current State Analysis

### What Exists
- ✅ Standalone kanban board at `/` with full drag-drop functionality
- ✅ Standalone map view at `/map` with incident markers
- ✅ Bidirectional navigation via URL params (`?highlight=<incident_id>`)
- ✅ Pulse animation for highlighted incidents on kanban board
- ✅ Map markers clickable with popups and details modal
- ✅ Shared incident detail modal across both views

### What's Needed
- New `/combined` route with split-panel layout
- Resizable divider between kanban and map (60/40 default)
- Real-time synchronization:
  - Hover kanban card → highlight map marker
  - Click map marker → highlight kanban card
- Kanban board without personnel/materials sidebars (space-saving)
- Full drag-drop functionality preserved in combined view
- Responsive handling for smaller screens

```typescript
// Dynamically import map to avoid SSR issues with Leaflet
const MapView = dynamic(() => import("@/components/map-view"), {

    <div className="w-full h-full flex items-center justify-center bg-zinc-900 rounded-lg">

    </div>
  ),

// Dynamically import kanban to avoid SSR drag-drop issues
const KanbanBoard = dynamic(() => import("@/components/combined-kanban-board"), {
  ssr: false,
  loading: () => (
    <div className="w-full h-full flex items-center justify-center bg-background">
      <div className="text-muted-foreground">Kanban wird geladen...</div>
    </div>
  ),
})

export default function CombinedViewPage() {
  const { incidents, refreshIncidents } = useIncidents()
  const [selectedIncidentId, setSelectedIncidentId] = useState<string | null>(null)
  const [hoveredIncidentId, setHoveredIncidentId] = useState<string | null>(null)
  const [incidentForModal, setIncidentForModal] = useState<Incident | null>(null)
  const [formOpen, setFormOpen] = useState(false)

  // Refresh incidents on mount
  useEffect(() => {
    refreshIncidents()
  }, [])

  // Active incidents (excluding completed)
  const activeIncidents = incidents.filter((inc) => inc.status !== "abschluss")

  const handleDetailsClick = (incident: Incident) => {
    setIncidentForModal(incident)
    setFormOpen(true)
  }

  const handleMapMarkerClick = (incidentId: string) => {
    setSelectedIncidentId(incidentId)
    // Trigger pulse animation on kanban card
    setTimeout(() => setSelectedIncidentId(null), 3000)
  }

  const handleKanbanCardHover = (incidentId: string | null) => {
    setHoveredIncidentId(incidentId)
  }

  return (
    <ProtectedRoute>
      <div className="flex h-screen flex-col bg-background text-foreground">
        {/* Header */}
        <header className="flex items-center justify-between border-b border-border/50 bg-card/50 backdrop-blur-sm px-6 py-4">
          <div className="flex items-center gap-3">
            <div className="flex h-11 w-11 items-center justify-center rounded-xl bg-gradient-to-br from-red-600 to-orange-600 text-2xl shadow-lg">
              🚒
            </div>
            <h1 className="text-2xl font-bold tracking-tight">Kombinierte Ansicht</h1>
            <Badge variant="secondary" className="ml-2">
              {activeIncidents.length} Aktiv
            </Badge>
          </div>

          <div className="flex items-center gap-4">
            <PageNavigation currentPage="combined" />
          </div>
        </header>

        {/* Main Content - Resizable Split View */}
        <main className="flex-1 overflow-hidden p-4">
          <PanelGroup direction="horizontal" className="h-full gap-4">
            {/* Kanban Board Panel - 60% default */}
            <Panel defaultSize={60} minSize={30}>
              <div className="h-full rounded-lg border border-border/50 bg-card/30 backdrop-blur-sm overflow-hidden">
                <KanbanBoard
                  onCardHover={handleKanbanCardHover}
                  onCardClick={handleDetailsClick}
                  highlightedIncidentId={selectedIncidentId}
                />
              </div>
            </Panel>

            {/* Resize Handle */}
            <PanelResizeHandle className="w-2 bg-border/50 hover:bg-primary/50 transition-colors rounded-full flex items-center justify-center group">
              <GripVertical className="h-4 w-4 text-muted-foreground group-hover:text-primary transition-colors" />
            </PanelResizeHandle>

            {/* Map Panel - 40% default */}
            <Panel defaultSize={40} minSize={25}>
              <div className="h-full rounded-lg border border-border/50 overflow-hidden">
                <MapView
                  selectedIncidentId={hoveredIncidentId}
                  onMarkerClick={handleMapMarkerClick}
                  onDetailsClick={handleDetailsClick}
                />
              </div>
            </Panel>
          </PanelGroup>
        </main>

        {/* Incident Form Modal */}
        <IncidentForm
          open={formOpen}
          onOpenChange={setFormOpen}
          incident={incidentForModal}
          mode="edit"
        />
      </div>
    </ProtectedRoute>
  )
}
```

### 3. Extract Kanban Board Component (No Sidebars)

**File**: `frontend/components/combined-kanban-board.tsx`

This component extracts the core kanban board logic from `app/page.tsx` but removes the personnel/materials sidebars to save space.

```typescript
"use client"

import { useState, useEffect, useCallback, useRef } from "react"
import { Card } from "@/components/ui/card"
import { Badge } from "@/components/ui/badge"
import { Clock, Users, Package, X, MapPin } from 'lucide-react'
import { useOperations } from "@/lib/contexts/operations-context"
import type { Operation } from "@/lib/contexts/operations-context"
// Import drag-drop utilities
import { draggable, dropTargetForElements } from '@atlaskit/pragmatic-drag-and-drop/element/adapter'
import { combine } from '@atlaskit/pragmatic-drag-and-drop/combine'
import { attachClosestEdge, extractClosestEdge, type Edge } from '@atlaskit/pragmatic-drag-and-drop-hitbox/closest-edge'
import { DropIndicator } from '@atlaskit/pragmatic-drag-and-drop-react-drop-indicator/box'

const columns = [
  { id: "incoming", title: "EINGEGANGEN", status: ["incoming"], color: "bg-zinc-800/50" },
  { id: "ready", title: "REKO", status: ["ready"], color: "bg-green-800/30" },
  { id: "enroute", title: "DISPONIERT / UNTERWEGS", status: ["enroute"], color: "bg-blue-900/30" },
  { id: "active", title: "EINSATZ", status: ["active"], color: "bg-orange-900/30" },
  { id: "returning", title: "BEENDET / RÜCKFAHRT", status: ["returning"], color: "bg-blue-800/30" },
  { id: "complete", title: "ABGESCHLOSSEN", status: ["complete"], color: "bg-zinc-900/50" },
]

// Helper function to get time since incident
function getTimeSince(date: Date): string {
  const minutes = Math.floor((Date.now() - date.getTime()) / 1000 / 60)
  if (minutes < 60) return `${minutes}'`
  const hours = Math.floor(minutes / 60)
  const mins = minutes % 60
  return `${hours}h ${mins}'`
}

interface CombinedKanbanBoardProps {
  onCardHover: (incidentId: string | null) => void
  onCardClick: (operation: Operation) => void
  highlightedIncidentId: string | null
}

export default function CombinedKanbanBoard({
  onCardHover,
  onCardClick,
  highlightedIncidentId
}: CombinedKanbanBoardProps) {
  const { operations, materials, formatLocation, removeCrew, removeMaterial } = useOperations()
  const [isMounted, setIsMounted] = useState(false)
  const isDraggingRef = useRef(false)

  useEffect(() => {
    setIsMounted(true)
  }, [])

  // DraggableOperation component (simplified - copy from page.tsx)
  function DraggableOperation({
    operation,
    columnColor,
    index,
  }: {
    operation: Operation
    columnColor: string
    index: number
  }) {
    const ref = useRef<HTMLDivElement>(null)
    const [isDragging, setIsDragging] = useState(false)
    const [closestEdge, setClosestEdge] = useState<Edge | null>(null)

    useEffect(() => {
      const element = ref.current
      if (!element) return

      return combine(
        draggable({
          element,
          getInitialData: () => ({ type: "operation", operation, index }),
          onDragStart: () => {
            setIsDragging(true)
            isDraggingRef.current = true
          },
          onDrop: () => {
            setIsDragging(false)
            setTimeout(() => {
              isDraggingRef.current = false
            }, 200)
          },
        }),
        dropTargetForElements({
          element,
          getData: ({ input }) => {
            return attachClosestEdge(
              { type: "operation-drop", operationId: operation.id, index },
              { element, input, allowedEdges: ['top', 'bottom'] }
            )
          },
          onDragEnter: ({ self }) => {
            const edge = extractClosestEdge(self.data)
            setClosestEdge(edge)
          },
          onDrag: ({ self }) => {
            const edge = extractClosestEdge(self.data)
            setClosestEdge(edge)
          },
          onDragLeave: () => {
            setClosestEdge(null)
          },
          onDrop: () => {
            setClosestEdge(null)
          },
        })
      )
    }, [operation, index])

    return (
      <div className="relative w-full">
        {closestEdge === 'top' && <DropIndicator edge="top" gap="4px" />}
        <Card
          ref={ref}
          style={{ opacity: isDragging ? 0.5 : 1 }}
          className={`${columnColor} border border-border/50 backdrop-blur-sm p-3 transition-all hover:border-primary/50 hover:shadow-lg cursor-pointer ${
            highlightedIncidentId === operation.id ? "ring-4 ring-accent animate-pulse" : ""
          }`}
          onMouseEnter={() => onCardHover(operation.id)}
          onMouseLeave={() => onCardHover(null)}
          onClick={() => {
            if (!isDraggingRef.current) {
              onCardClick(operation)
            }
          }}
        >
          <div className="space-y-2">
            <div className="flex items-start gap-2">
              <div
                className={`h-2 w-2 rounded-full flex-shrink-0 mt-1 ${
                  operation.priority === "high"
                    ? "bg-red-500"
                    : operation.priority === "medium"
                    ? "bg-yellow-500"
                    : "bg-green-500"
                }`}
              />
              <h3 className="font-bold text-sm leading-tight flex-1">
                {formatLocation(operation.location)}
              </h3>
            </div>

            <div className="flex items-center justify-between text-xs">
              <span className="font-mono text-muted-foreground">
                {operation.dispatchTime.toLocaleTimeString("de-DE", {
                  hour: "2-digit",
                  minute: "2-digit",
                })}
              </span>
              <span className="font-mono text-muted-foreground">
                {getTimeSince(operation.statusChangedAt || operation.dispatchTime)}
              </span>
            </div>

            {operation.crew.length > 0 && (
              <div className="flex items-center gap-1.5 text-xs">
                <Users className="h-3 w-3 text-muted-foreground" />
                <div className="flex flex-wrap gap-1">
                  {operation.crew.slice(0, 3).map((member, idx) => (
                    <Badge key={idx} variant="secondary" className="text-xs px-1.5 py-0">
                      {member.split(" ")[0][0]}.{member.split(" ")[1][0]}
                    </Badge>
                  ))}
                  {operation.crew.length > 3 && (
                    <Badge variant="secondary" className="text-xs px-1.5 py-0">
                      +{operation.crew.length - 3}
                    </Badge>
                  )}
                </div>
              </div>
            )}
          </div>
        </Card>
        {closestEdge === 'bottom' && <DropIndicator edge="bottom" gap="4px" />}
      </div>
    )
  }

  // DroppableColumn component (simplified)
  function DroppableColumn({ column, operations }: { column: typeof columns[0]; operations: Operation[] }) {
    return (
      <div className="flex w-64 flex-shrink-0 flex-col">
        <div className={`mb-2 rounded-lg ${column.color} border border-border/50 px-3 py-2`}>
          <h2 className="text-xs font-bold uppercase tracking-wide text-foreground">
            {column.title}
          </h2>
          <p className="text-xs text-muted-foreground">{operations.length}</p>
        </div>

        <div className="flex-1 space-y-2 overflow-y-auto p-1">
          {operations.map((operation, index) => (
            <DraggableOperation
              key={operation.id}
              operation={operation}
              columnColor={column.color}
              index={index}
            />
          ))}
        </div>
      </div>
    )
  }

  if (!isMounted) {
    return (
      <div className="flex h-full items-center justify-center">
        <div className="text-muted-foreground">Laden...</div>
      </div>
    )
  }

  return (
    <div className="flex h-full gap-3 overflow-x-auto p-4 bg-zinc-950/20">
      {columns.map((column) => {
        const columnOps = operations.filter((op) => column.status.includes(op.status))
        return <DroppableColumn key={column.id} column={column} operations={columnOps} />
      })}
    </div>
  )
}
```

## Responsive Behavior

**Large Screens (≥1920px)**:
- Side-by-side split view works perfectly
- Default 60/40 split
- Resizable divider active

**Medium Screens (1280px - 1919px)**:
- Side-by-side still works but may be cramped
- User can resize to preferred ratio
- Consider 70/30 or 50/50 if needed

**Small Screens (<1280px)**:
- **Option 1**: Stack vertically (kanban on top, map below)
- **Option 2**: Show warning "Combined view works best on larger screens"
- **Option 3**: User accepts reduced usability (scrolling required)

**Recommendation**: Start with Option 3 (let user deal with it), implement Option 1 if complaints arise.

## Security Considerations

- All incident data displayed in combined view respects role-based permissions
- Editor vs. Viewer roles enforced (no additional security needed)
- No new API endpoints required (uses existing incident API)

## Performance Notes

### Optimization Strategies
- **Dynamic imports**: Prevent SSR issues with Leaflet and drag-drop
- **Memoization**: Use React.memo for DraggableOperation components
- **Virtualization**: If >50 incidents per column, consider virtual scrolling
- **Debounce hover**: Avoid excessive re-renders on fast mouse movement

### Monitoring
- Track combined view usage in analytics
- Monitor page load time (target: <2s)
- Watch for increased memory usage (two complex views simultaneously)
