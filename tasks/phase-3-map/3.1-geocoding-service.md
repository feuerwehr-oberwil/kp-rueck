### 1. Create Geocoding Service

**File**: `backend/app/services/geocoding.py`

```python
"""
Geocoding service using Nominatim (OpenStreetMap) API.
Provides address-to-coordinates conversion with caching.
"""

import asyncio
import logging
from typing import Optional, Tuple, Dict, Any
from datetime import datetime, timedelta
import aiohttp
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import select, insert, update

from app.config import get_settings

logger = logging.getLogger(__name__)

# In-memory cache for geocoding results (expires after 24 hours)
_geocode_cache: Dict[str, Tuple[float, float, datetime]] = {}
CACHE_EXPIRY = timedelta(hours=24)


class GeocodingService:
    """Service for geocoding addresses using Nominatim API."""

    def __init__(self):
        self.settings = get_settings()
        self.base_url = "https://nominatim.openstreetmap.org"
        self.user_agent = "KP-Rueck-Feuerwehr-Oberwil/1.0"
        # Rate limiting: 1 request per second for Nominatim
        self._last_request_time = 0.0
        self._rate_limit_delay = 1.0

    async def _rate_limit(self):
        """Enforce rate limiting (1 req/sec for Nominatim)."""
        now = asyncio.get_event_loop().time()
        time_since_last = now - self._last_request_time
        if time_since_last < self._rate_limit_delay:
            await asyncio.sleep(self._rate_limit_delay - time_since_last)
        self._last_request_time = asyncio.get_event_loop().time()

    def _check_cache(self, address: str) -> Optional[Tuple[float, float]]:
        """Check in-memory cache for geocoding result."""
        normalized_address = self._normalize_address(address)
        if normalized_address in _geocode_cache:
            lat, lng, cached_at = _geocode_cache[normalized_address]
            if datetime.now() - cached_at < CACHE_EXPIRY:
                logger.info(f"Cache hit for address: {address}")
                return (lat, lng)
            else:
                # Expired, remove from cache
                del _geocode_cache[normalized_address]
        return None

    def _cache_result(self, address: str, lat: float, lng: float):
        """Store geocoding result in cache."""
        normalized_address = self._normalize_address(address)
        _geocode_cache[normalized_address] = (lat, lng, datetime.now())

    def _normalize_address(self, address: str) -> str:
        """Normalize address for consistent caching."""
        return address.lower().strip().replace("  ", " ")

    async def geocode_address(
        self,
        address: str,
        country: str = "Switzerland"
    ) -> Optional[Dict[str, Any]]:
        """
        Convert address to coordinates using Nominatim API.

        Args:
            address: The address to geocode
            country: Country to limit search (default: Switzerland)

        Returns:
            Dict with keys: lat, lng, display_name, or None if not found
        """
        # Check cache first
        cached = self._check_cache(address)
        if cached:
            return {
                "lat": cached[0],
                "lng": cached[1],
                "display_name": address,
                "source": "cache"
            }

        # Rate limit before API call
        await self._rate_limit()

        params = {
            "q": address,
            "format": "json",
            "addressdetails": 1,
            "limit": 1,
            "countrycodes": "ch" if country == "Switzerland" else None
        }

        try:
            async with aiohttp.ClientSession() as session:
                async with session.get(
                    f"{self.base_url}/search",
                    params=params,
                    headers={"User-Agent": self.user_agent},
                    timeout=aiohttp.ClientTimeout(total=10)
                ) as response:
                    if response.status != 200:
                        logger.error(f"Nominatim API error: {response.status}")
                        return None

                    data = await response.json()

                    if not data or len(data) == 0:
                        logger.warning(f"No results found for address: {address}")
                        return None

                    result = data[0]
                    lat = float(result["lat"])
                    lng = float(result["lon"])
                    display_name = result.get("display_name", address)

                    # Validate coordinates (Switzerland bounds approximately)
                    if not self.validate_coordinates(lat, lng):
                        logger.warning(
                            f"Coordinates outside expected bounds: {lat}, {lng}"
                        )
                        return None

                    # Cache the result
                    self._cache_result(address, lat, lng)

                    return {
                        "lat": lat,
                        "lng": lng,
                        "display_name": display_name,
                        "source": "nominatim"
                    }

        except asyncio.TimeoutError:
            logger.error(f"Geocoding timeout for address: {address}")
            return None
        except Exception as e:
            logger.error(f"Geocoding error: {str(e)}")
            return None

    async def reverse_geocode(
        self,
        lat: float,
        lng: float
    ) -> Optional[str]:
        """
        Convert coordinates to address using reverse geocoding.

        Args:
            lat: Latitude
            lng: Longitude

        Returns:
            Address string or None if not found
        """
        if not self.validate_coordinates(lat, lng):
            logger.warning(f"Invalid coordinates: {lat}, {lng}")
            return None

        await self._rate_limit()

        params = {
            "lat": lat,
            "lon": lng,
            "format": "json",
            "addressdetails": 1
        }

        try:
            async with aiohttp.ClientSession() as session:
                async with session.get(
                    f"{self.base_url}/reverse",
                    params=params,
                    headers={"User-Agent": self.user_agent},
                    timeout=aiohttp.ClientTimeout(total=10)
                ) as response:
                    if response.status != 200:
                        logger.error(f"Reverse geocoding error: {response.status}")
                        return None

                    data = await response.json()
                    return data.get("display_name")

        except Exception as e:
            logger.error(f"Reverse geocoding error: {str(e)}")
            return None

    def validate_coordinates(
        self,
        lat: float,
        lng: float,
        region: str = "switzerland"
    ) -> bool:
        """
        Validate that coordinates are within expected bounds.

        Args:
            lat: Latitude
            lng: Longitude
            region: Geographic region for bounds checking

        Returns:
            True if coordinates are valid and within bounds
        """
        # Switzerland approximate bounds
        if region.lower() == "switzerland":
            return (45.8 <= lat <= 47.8) and (5.9 <= lng <= 10.5)

        # Global bounds
        return (-90 <= lat <= 90) and (-180 <= lng <= 180)

    async def search_addresses(
        self,
        query: str,
        limit: int = 5
    ) -> list[Dict[str, Any]]:
        """
        Search for addresses matching query (for autocomplete).

        Args:
            query: Search query
            limit: Maximum number of results

        Returns:
            List of address suggestions
        """
        await self._rate_limit()

        params = {
            "q": query,
            "format": "json",
            "addressdetails": 1,
            "limit": limit,
            "countrycodes": "ch"
        }

        try:
            async with aiohttp.ClientSession() as session:
                async with session.get(
                    f"{self.base_url}/search",
                    params=params,
                    headers={"User-Agent": self.user_agent},
                    timeout=aiohttp.ClientTimeout(total=10)
                ) as response:
                    if response.status != 200:
                        return []

                    data = await response.json()

                    return [
                        {
                            "display_name": item["display_name"],
                            "lat": float(item["lat"]),
                            "lng": float(item["lon"]),
                            "type": item.get("type", "unknown")
                        }
                        for item in data
                    ]

        except Exception as e:
            logger.error(f"Address search error: {str(e)}")
            return []


# Global service instance
_geocoding_service: Optional[GeocodingService] = None


def get_geocoding_service() -> GeocodingService:
    """Get or create the global geocoding service instance."""
    global _geocoding_service
    if _geocoding_service is None:
        _geocoding_service = GeocodingService()
    return _geocoding_service
```

"""
Geocoding API endpoints.
"""

from typing import Optional
from fastapi import APIRouter, HTTPException, Query
from pydantic import BaseModel, Field

from app.services.geocoding import get_geocoding_service

router = APIRouter(prefix="/geocoding", tags=["geocoding"])


class GeocodeRequest(BaseModel):
    """Request body for geocoding."""
    address: str = Field(..., min_length=3, max_length=500)
    country: Optional[str] = Field(default="Switzerland")


class GeocodeResponse(BaseModel):
    """Response for geocoding."""
    lat: float
    lng: float
    display_name: str
    source: str  # "cache" or "nominatim"


class ReverseGeocodeResponse(BaseModel):
    """Response for reverse geocoding."""
    address: str


class AddressSuggestion(BaseModel):
    """Address suggestion for autocomplete."""
    display_name: str
    lat: float
    lng: float
    type: str


@router.post("/geocode", response_model=GeocodeResponse)
async def geocode_address(request: GeocodeRequest):
    """
    Convert address to coordinates.

    Example request:
    ```json
    {
        "address": "Hauptstrasse 10, 4104 Oberwil",
        "country": "Switzerland"
    }
    ```
    """
    service = get_geocoding_service()
    result = await service.geocode_address(request.address, request.country)

    if result is None:
        raise HTTPException(
            status_code=404,
            detail=f"Address not found: {request.address}"
        )

    return result


@router.get("/reverse", response_model=ReverseGeocodeResponse)
async def reverse_geocode(
    lat: float = Query(..., ge=-90, le=90),
    lng: float = Query(..., ge=-180, le=180)
):
    """
    Convert coordinates to address.

    Example: GET /api/geocoding/reverse?lat=47.5149&lng=7.5177
    """
    service = get_geocoding_service()
    address = await service.reverse_geocode(lat, lng)

    if address is None:
        raise HTTPException(
            status_code=404,
            detail=f"No address found for coordinates: {lat}, {lng}"
        )

    return {"address": address}


@router.get("/search", response_model=list[AddressSuggestion])
async def search_addresses(
    q: str = Query(..., min_length=3, max_length=200),
    limit: int = Query(default=5, ge=1, le=10)
):
    """
    Search for addresses (for autocomplete).

    Example: GET /api/geocoding/search?q=Hauptstrasse&limit=5
    """
    service = get_geocoding_service()
    results = await service.search_addresses(q, limit)
    return results
```

```typescript
'use client'

import { useState, useEffect, useRef } from 'react'
import { Input } from '@/components/ui/input'
import { Label } from '@/components/ui/label'
import { MapPin, Loader2 } from 'lucide-react'
import { apiClient } from '@/lib/api-client'
import { cn } from '@/lib/utils'

interface AddressSuggestion {
  display_name: string
  lat: number
  lng: number
  type: string
}

interface AddressAutocompleteProps {
  value: string
  onChange: (address: string, lat?: number, lng?: number) => void
  label?: string
  placeholder?: string
  required?: boolean
  className?: string
}

export default function AddressAutocomplete({
  value,
  onChange,
  label = "Address",
  placeholder = "Start typing address...",
  required = false,
  className
}: AddressAutocompleteProps) {
  const [query, setQuery] = useState(value)
  const [suggestions, setSuggestions] = useState<AddressSuggestion[]>([])
  const [isLoading, setIsLoading] = useState(false)
  const [showSuggestions, setShowSuggestions] = useState(false)
  const [selectedIndex, setSelectedIndex] = useState(-1)
  const debounceTimer = useRef<NodeJS.Timeout>()
  const wrapperRef = useRef<HTMLDivElement>(null)

  // Close suggestions when clicking outside
  useEffect(() => {
    function handleClickOutside(event: MouseEvent) {
      if (wrapperRef.current && !wrapperRef.current.contains(event.target as Node)) {
        setShowSuggestions(false)
      }
    }

    document.addEventListener('mousedown', handleClickOutside)
    return () => document.removeEventListener('mousedown', handleClickOutside)
  }, [])

  // Update local state when value prop changes
  useEffect(() => {
    setQuery(value)
  }, [value])

  async function searchAddresses(searchQuery: string) {
    if (searchQuery.length < 3) {
      setSuggestions([])
      return
    }

    setIsLoading(true)

    try {
      const results = await apiClient.searchAddresses(searchQuery, 5)
      setSuggestions(results)
      setShowSuggestions(true)
      setSelectedIndex(-1)
    } catch (error) {
      console.error('Address search failed:', error)
      setSuggestions([])
    } finally {
      setIsLoading(false)
    }
  }

  function handleInputChange(e: React.ChangeEvent<HTMLInputElement>) {
    const newQuery = e.target.value
    setQuery(newQuery)

    // Clear existing timer
    if (debounceTimer.current) {
      clearTimeout(debounceTimer.current)
    }

    // Debounce search (300ms)
    debounceTimer.current = setTimeout(() => {
      searchAddresses(newQuery)
    }, 300)
  }

  function handleSelectSuggestion(suggestion: AddressSuggestion) {
    setQuery(suggestion.display_name)
    onChange(suggestion.display_name, suggestion.lat, suggestion.lng)
    setSuggestions([])
    setShowSuggestions(false)
  }

  function handleKeyDown(e: React.KeyboardEvent) {
    if (!showSuggestions || suggestions.length === 0) return

    switch (e.key) {
      case 'ArrowDown':
        e.preventDefault()
        setSelectedIndex(prev =>
          prev < suggestions.length - 1 ? prev + 1 : prev
        )
        break
      case 'ArrowUp':
        e.preventDefault()
        setSelectedIndex(prev => prev > 0 ? prev - 1 : -1)
        break
      case 'Enter':
        e.preventDefault()
        if (selectedIndex >= 0) {
          handleSelectSuggestion(suggestions[selectedIndex])
        }
        break
      case 'Escape':
        setShowSuggestions(false)
        break
    }
  }

  return (
    <div ref={wrapperRef} className={cn("relative", className)}>
      <Label htmlFor="address">
        {label} {required && '*'}
      </Label>
      <div className="relative mt-1">
        <MapPin className="absolute left-3 top-1/2 -translate-y-1/2 h-4 w-4 text-muted-foreground" />
        <Input
          id="address"
          type="text"
          value={query}
          onChange={handleInputChange}
          onKeyDown={handleKeyDown}
          onFocus={() => {
            if (suggestions.length > 0) {
              setShowSuggestions(true)
            }
          }}
          placeholder={placeholder}
          required={required}
          className="pl-9 pr-9"
        />
        {isLoading && (
          <Loader2 className="absolute right-3 top-1/2 -translate-y-1/2 h-4 w-4 animate-spin text-muted-foreground" />
        )}
      </div>

      {showSuggestions && suggestions.length > 0 && (
        <div className="absolute z-50 w-full mt-1 bg-background border rounded-md shadow-lg max-h-60 overflow-auto">
          {suggestions.map((suggestion, index) => (
            <button
              key={index}
              type="button"
              className={cn(
                "w-full text-left px-3 py-2 hover:bg-accent transition-colors",
                "flex items-start gap-2",
                selectedIndex === index && "bg-accent"
              )}
              onClick={() => handleSelectSuggestion(suggestion)}
            >
              <MapPin className="h-4 w-4 mt-0.5 text-muted-foreground flex-shrink-0" />
              <div className="flex-1 min-w-0">
                <div className="text-sm truncate">{suggestion.display_name}</div>
                <div className="text-xs text-muted-foreground">
                  {suggestion.type}
                </div>
              </div>
            </button>
          ))}
        </div>
      )}
    </div>
  )
}
```

## Security Considerations

### API Rate Limiting
- Enforce 1 request per second to Nominatim (public API terms)
- Implement internal rate limiting to prevent abuse
- Log excessive geocoding requests per user

### Input Validation
- Validate address length (3-500 characters)
- Sanitize address input to prevent injection attacks
- Validate coordinate ranges before database storage

### Authentication
- All geocoding endpoints require valid JWT token
- Both Editor and Viewer roles can access geocoding
- No public unauthenticated geocoding access

### Data Privacy
- Geocoding requests logged but not stored permanently
- No personal data sent to Nominatim API
- Cache keys normalized to prevent data leakage

## Performance Notes

### Optimization Strategies
- **In-memory caching**: 24-hour cache for geocoding results
- **Rate limiting**: Prevent API overload (1 req/sec)
- **Debounced search**: Wait 300ms before autocomplete API call
- **Async operations**: Non-blocking geocoding during incident creation
- **Timeout handling**: 10-second timeout for API calls
- **Fallback gracefully**: Failed geocoding doesn't block incident creation

### Monitoring
- Log geocoding API response times
- Track cache hit rate
- Monitor rate limit violations
- Alert on repeated geocoding failures

### Scalability
- In-memory cache suitable for single-instance deployment
- For multi-instance: Consider Redis cache
- Nominatim public API sufficient for MVP (<1000 req/day expected)
- Consider self-hosted Nominatim if usage exceeds limits

## Migration Path

If starting fresh:
1. Implement geocoding service with caching

4. Integrate with incident creation
5. Test with real Swiss addresses

If upgrading existing system:
1. Deploy geocoding service without breaking existing functionality

3. Backfill existing incidents with coordinates (migration script)
4. Monitor API usage and cache performance
5. Gradually enable auto-geocoding features
