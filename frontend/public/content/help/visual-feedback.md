# Visuelle Hinweise & Bedienelemente

Verstehen Sie die visuellen Hinweise und Bedienelemente des Systems für effiziente Einsatzführung.

## Prioritätsanzeigen

Einsätze werden farblich nach ihrer Priorität gekennzeichnet, um schnelles Erfassen der Dringlichkeit zu ermöglichen.

### Farbcodierung

**Kritisch (Rot)**
- Sofortige Reaktion erforderlich
- Rote Umrandung der Einsatzkarte
- Höchste Dringlichkeit

**Hoch (Orange)**
- Dringende Aufmerksamkeit notwendig
- Orange Umrandung
- Zeitkritische Bearbeitung

**Mittel (Gelb)**
- Standard-Priorität
- Gelbe Umrandung
- Normale Bearbeitungsreihenfolge

**Niedrig (Grau)**
- Kann warten
- Graue Umrandung
- Niedrigste Dringlichkeit

### Priorität ändern

**Per Tastatur:**
- Einsatz mit Maus auswählen
- `Shift + 1` = Niedrige Priorität
- `Shift + 2` = Mittlere Priorität
- `Shift + 3` = Hohe Priorität
- `Shift + 4` = Kritische Priorität

**Per Dialog:**
- Einsatz-Details öffnen (`E` oder `Enter`)
- Priorität im Formular ändern
- Speichern

## Alters-Indikatoren

Farbliche Kennzeichnung zeigt, wie lange ein Einsatz bereits aktiv ist.

### Badge-Farben

**Grün "Neu"**
- Einsatz ist weniger als 15 Minuten alt
- Frisch eingegangen
- Noch in der Erstbearbeitung

**Gelb**
- Aktiver Einsatz (15-60 Minuten)
- Laufende Bearbeitung
- Normale Bearbeitungszeit

**Orange**
- Älter werdender Einsatz (1-2 Stunden)
- Längere Bearbeitung
- Aufmerksamkeit empfohlen

**Rot mit Warnsymbol**
- Alter Einsatz (über 2 Stunden)
- Benötigt Aufmerksamkeit
- Überprüfung erforderlich

### Genaue Zeit anzeigen

Bewegen Sie die Maus über den Alters-Badge, um die exakte Erstellungszeit zu sehen (Tooltip).

## Ressourcen-Status

Schneller Überblick über zugewiesene Ressourcen direkt auf der Einsatzkarte.

### Status-Symbole

**Grünes Häkchen (✓)**
- Ressourcen sind zugewiesen
- Zeigt Anzahl der zugewiesenen Einheiten
- Beispiel: "Mannschaft (3)" mit ✓

**Graues X (❌)**
- Keine Ressourcen zugewiesen
- Benötigt Zuweisung
- Beispiel: "Fahrzeuge (0)" mit ❌

### Ressourcen-Kategorien

**Mannschaft**
- Anzahl zugewiesener Personen
- Klick auf [+] zum Zuweisen
- Drag & Drop aus linker Seitenleiste

**Fahrzeuge**
- Anzahl zugewiesener Fahrzeuge
- Klick auf [+] zum Zuweisen
- Drag & Drop aus rechter Seitenleiste

**Material**
- Anzahl zugewiesener Materialien
- Klick auf [+] zum Zuweisen
- Drag & Drop aus rechter Seitenleiste

## Drag & Drop Bedienung

### Desktop (Maus)

**Ressource ziehen:**
1. Maus über ziehbares Element bewegen
2. Cursor ändert sich zu Hand-Symbol
3. Klicken und halten
4. Element wird leicht transparent und kleiner
5. Über Zielkarte bewegen
6. Ziel-Bereich wird hervorgehoben
7. Loslassen zum Zuweisen

**Visuelle Hinweise beim Ziehen:**
- **Grab-Cursor**: Element kann gezogen werden
- **Grabbing-Cursor**: Element wird gerade gezogen
- **Skalierung**: Gezogenes Element wird 95% der Originalgröße
- **Transparenz**: 80% Deckkraft während des Ziehens
- **Rahmen**: Primärfarbe zeigt aktives Ziehen
- **Drop-Zone**: Gestrichelte Umrandung bei gültigem Ziel

**Einsatz verschieben:**
1. Einsatzkarte anklicken und halten
2. In andere Status-Spalte ziehen
3. Spalte wird hervorgehoben
4. Loslassen zum Verschieben

### Mobile & Tablet (Touch)

**Klick-zu-Zuweisen Methode:**
1. Auf [+] Button bei Ressourcen-Kategorie klicken
2. Dialog öffnet sich mit verfügbaren Ressourcen
3. Ressource auswählen (einzeln antippen)
4. Häkchen zeigt Zuweisung an
5. "Fertig" klicken zum Schließen

**Vorteile:**
- Einfacher auf Touch-Geräten
- Keine präzise Zieh-Bewegung nötig
- Suchfunktion für schnelles Finden
- Klare Bestätigung der Zuweisung

**Einsatz verschieben:**
1. Einsatzkarte gedrückt halten (ca. 500ms)
2. Ziehen in andere Spalte
3. Loslassen zum Verschieben

### Wann welche Methode?

**Drag & Drop verwenden:**
- Desktop mit Maus
- Mehrere Ressourcen schnell zuweisen
- Visuelles Arbeiten bevorzugt

**Klick-zu-Zuweisen verwenden:**
- Tablet oder Smartphone
- Suche nach spezifischer Ressource
- Präzise einzelne Zuweisung

## Tastaturkürzel

Vollständige Liste aller Tastaturkürzel: Drücken Sie `?` oder siehe [Tastaturkürzel](/help/keyboard-shortcuts)

### Wichtigste Shortcuts

**Navigation:**
- `G` + `K` = Kanban Board
- `G` + `M` = Lagekarte
- `G` + `E` = Ereignisse

**Aktionen:**
- `N` = Neuer Einsatz
- `/` = Suche fokussieren
- `R` = Aktualisieren
- `?` = Tastaturkürzel-Hilfe

**Einsatz bearbeiten** (Einsatz zuerst auswählen):
- `E` oder `Enter` = Details öffnen
- `1-5` = Fahrzeug zuweisen/entfernen
- `Shift+1-4` = Priorität ändern
- `<` oder `,` = Status zurück
- `>` oder `.` = Status weiter

**Einsatz-Navigation:**
- `↑` = Vorheriger Einsatz
- `↓` = Nächster Einsatz
- `Tab` = Durchlaufen
- `[` = Linke Sidebar ein/aus
- `]` = Rechte Sidebar ein/aus

## Tipps für effiziente Bedienung

**Desktop-Nutzer:**
- Lernen Sie die wichtigsten Tastaturkürzel auswendig
- Nutzen Sie Drag & Drop für schnelle Zuweisungen
- Verwenden Sie `Cmd+K` (Mac) oder `Ctrl+K` (Windows) für Befehlspalette

**Mobile-Nutzer:**
- Nutzen Sie den [+] Button für Zuweisungen
- Verwenden Sie die Such-Funktion in Dialogen
- Bottom-Navigation für schnellen Ansichtswechsel

**Profi-Tipp:**
Bewegen Sie die Maus über einen Einsatz (oder wählen Sie ihn mit ↑/↓ aus), um ihn mit Tastenkombinationen zu bearbeiten. Der ausgewählte Einsatz wird hervorgehoben.

## Barrierefreiheit

**Tastatur-Navigation:**
- Alle Funktionen sind per Tastatur erreichbar
- Tab-Taste zum Durchlaufen von Elementen
- Enter zum Aktivieren von Buttons

**Bildschirm-Reader:**
- Beschriftungen für alle interaktiven Elemente
- ARIA-Labels für bessere Zugänglichkeit
- Semantisches HTML für klare Struktur

**Touch-Targets:**
- Mindestgröße 44px × 44px (WCAG AAA)
- Ausreichend Abstand zwischen Elementen
- Große Buttons für kritische Aktionen
