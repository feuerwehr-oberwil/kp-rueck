# Tastaturkürzel

KP Rück bietet umfassende Tastaturunterstützung für effizientes Arbeiten ohne Maus.

## Schnellreferenz

**Tipp:** Drücke `?` um die Schnellreferenz anzuzeigen.

## Navigation

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `G dann K` | Kanban-Ansicht öffnen |
| `G dann M` | Karten-Ansicht öffnen |
| `G dann E` | Ereignis-Auswahl öffnen |

## Aktionen

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `N` | Neuen Einsatz erstellen |
| `E oder Enter` | Ausgewählten Einsatz bearbeiten |
| `R oder F5` | Daten aktualisieren |
| `?` | Tastaturkürzel-Schnellreferenz anzeigen |

## Suche

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `/` | Einsätze durchsuchen (Fokus auf Suchfeld) |
| `P` | Personen durchsuchen |
| `M` | Material durchsuchen |
| `Esc` | Suchfeld verlassen / Auswahl aufheben |

## Einsatz-Navigation

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `↑ (Pfeil hoch)` | Vorherigen Einsatz auswählen |
| `↓ (Pfeil runter)` | Nächsten Einsatz auswählen |
| `Enter` | Ausgewählten Einsatz bearbeiten |
| `Delete oder Backspace` | Ausgewählten Einsatz löschen |

## Status-Änderung

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `> oder .` | Einsatz-Status vorwärts bewegen (nächste Spalte) |
| `< oder ,` | Einsatz-Status rückwärts bewegen (vorherige Spalte) |

## Fahrzeug-Zuweisung (im Kanban)

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `1` | Erstes Fahrzeug zuweisen/entfernen |
| `2` | Zweites Fahrzeug zuweisen/entfernen |
| `3` | Drittes Fahrzeug zuweisen/entfernen |
| `4` | Viertes Fahrzeug zuweisen/entfernen |
| `5` | Fünftes Fahrzeug zuweisen/entfernen |

## Priorität

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `Shift+1` | Priorität auf Niedrig setzen |
| `Shift+2` | Priorität auf Mittel setzen |
| `Shift+3` | Priorität auf Hoch setzen |

## Ansicht

| Tastenkürzel | Beschreibung |
|--------------|-------------|
| `[` | Personen-Seitenleiste ein-/ausblenden |
| `]` | Material-Seitenleiste ein-/ausblenden |

## Tipps

### Navigation mit G-Taste
Drücke `G` gefolgt von einem Buchstaben, um schnell zwischen Ansichten zu wechseln:
- `G` + `K` → Kanban
- `G` + `M` → Map
- `G` + `E` → Events

### Einsatz-Workflow
1. Drücke `N` um einen neuen Einsatz zu erstellen
2. Navigiere mit `↑`/`↓` durch die Einsätze
3. Drücke `Enter` um Details zu bearbeiten
4. Nutze `>` und `<` um den Status zu ändern
5. Nutze `1`-`5` um Fahrzeuge zuzuweisen

### Suche
- `/` öffnet die Hauptsuche für Einsätze
- Beginne einfach zu tippen, um zu filtern
- `Esc` um die Suche zu verlassen

## Häufig verwendete Kombinationen

| Aufgabe | Tastenkürzel |
|---------|--------------|
| Schnell einen Einsatz finden | `/` → tippen → `Enter` |
| Einsatz erstellen und bearbeiten | `N` → Formular ausfüllen |
| Zwischen Ansichten wechseln | `G` + `K`/`M`/`E` |
| Einsatz weiter bearbeiten | `↓`/`↑` → `Enter` |
| Status schnell ändern | Einsatz auswählen → `>` oder `<` |
